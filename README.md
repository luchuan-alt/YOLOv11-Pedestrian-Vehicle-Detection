\# YOLOv11 行人与车辆检测实验

基于 YOLOv11 的行人与车辆目标检测课程实验



\## 项目简介

本项目使用 YOLOv11 轻量级模型实现行人与车辆的实时检测，包含训练脚本、推理脚本、数据集配置文件，适用于人工智能/计算机视觉课程实验。



\## 项目文件

\- train.py        模型训练代码

\- infer.py        模型推理代码

\- data.yaml       数据集配置

\- README.md       项目说明



\## 环境配置

pip install ultralytics



\## 运行方式

训练：python train.py

推理：python infer.py
